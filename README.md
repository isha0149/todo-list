# To-Do List App

A simple, responsive to-do list application built with HTML, CSS, and JavaScript.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6+)

## How to Use
1. Clone this repository
2. Open `index.html` in your browser
3. Start adding your tasks!
